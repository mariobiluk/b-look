7. Algoritmo de Busca Linear

Implementação e explicação do algoritmo de busca linear em arrays.

Explicação: Algoritmo de Busca Linear

O que é Busca Linear?

A busca linear, também conhecida como busca sequencial, é um algoritmo simples de busca que verifica cada elemento de uma lista sequencialmente até encontrar o elemento desejado ou percorrer toda a lista.

Este algoritmo é intuitivo e fácil de implementar, mas pode ser ineficiente para listas grandes, pois no pior caso precisa verificar todos os elementos. Sua complexidade é O(n), onde n é o número de elementos na lista.

Quando Usar Busca Linear?

  • Listas pequenas ou não ordenadas
  • Quando a simplicidade é mais importante que a eficiência
  • Como introdução ao conceito de algoritmos de busca

Pseudocódigo da Busca Linear

função busca_linear(lista, elemento):
    para i de 0 até tamanho(lista)-1:
        se lista[i] == elemento:
            retornar i  // elemento encontrado
    retornar -1  // elemento não encontrado
