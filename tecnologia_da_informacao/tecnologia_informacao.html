Fundamentos de Tecnologia da Informação

A base que sustenta sistemas, inovações e o futuro digital

Tecnologia da Informação: O Alicerce do Mundo Digital

A Tecnologia da Informação (TI) engloba todos os recursos tecnológicos utilizados para coletar, armazenar, processar e transmitir informações. É a espinha dorsal das organizações modernas e da sociedade conectada em que vivemos.

1. Hardware de Computadores: Componentes Fundamentais

O que é Hardware?

Hardware refere-se a todas as partes físicas e tangíveis de um computador, incluindo componentes eletrônicos, circuitos, periféricos e dispositivos mecânicos.

Placa Mãe (Motherboard)

Função: Conectar e interligar todos os componentes do computador

Características:

  • Slot para processador
  • Slots de memória RAM
  • Portas de expansão (PCI, PCI Express)
  • Conectores SATA para armazenamento
  • Portas USB, áudio, rede

Processador (CPU)

Função: Cérebro do computador, executa instruções e processa dados

Características:

  • Velocidade de clock (GHz)
  • Número de núcleos
  • Cache L1, L2, L3
  • Arquitetura (x86, x64, ARM)

Principais Fabricantes: Intel, AMD

Memória Principal (RAM)

Função: Armazenamento temporário de dados em uso

Características:

  • Acesso rápido aos dados
  • Volátil (perde dados ao desligar)
  • Capacidade medida em GB
  • Velocidade medida em MHz

Tipos: DDR3, DDR4, DDR5

2. Sistema Operacional: Conceito e Exemplos

O que é um Sistema Operacional?

É o software mais importante do computador, que gerencia todos os recursos de hardware e fornece uma interface para os usuários e aplicativos.

Funções Principais

  • Gerenciamento de processos
  • Gerenciamento de memória
  • Controle de dispositivos de E/S
  • Sistema de arquivos
  • Interface com o usuário

Exemplos de Sistemas Operacionais

Windows

Desenvolvedor: Microsoft

Características: Interface gráfica amigável, ampla compatibilidade

Usos: Desktops, laptops, servidores

Linux

Desenvolvedor: Comunidade Open Source

Características: Open source, estável, seguro

Usos: Servidores, embedded, desktops

macOS

Desenvolvedor: Apple

Características: Interface elegante, integração com ecossistema Apple

Usos: Computadores Mac

Android

Desenvolvedor: Google

Características: Baseado em Linux, para dispositivos móveis

Usos: Smartphones, tablets, TVs

3. Sistema de Arquivos: Modo Texto vs Modo Gráfico

O que é Sistema de Arquivos?

É o método e estrutura de dados que um sistema operacional usa para controlar como os dados são armazenados e recuperados em um dispositivo de armazenamento.

Tipos Comuns

  • NTFS: Windows moderno
  • FAT32: Dispositivos removíveis
  • ext4: Linux
  • APFS: macOS

Modo Texto (CMD/Prompt de Comando)

Interface: Baseada em texto, comandos digitados

Vantagens: Leve, rápido, automatização via scripts

Comandos Básicos:

dir          # Lista arquivos e diretórios
cd pasta     # Entra em um diretório
mkdir nova   # Cria novo diretório
copy arquivo.txt destino  # Copia arquivo
del arquivo.txt           # Exclui arquivo

Modo Gráfico (Windows Explorer)

Interface: Gráfica, ícones, menus, arrastar e soltar

Vantagens: Intuitiva, visual, fácil para usuários finais

Operações Básicas:

  • Duplo clique para abrir
  • Arrastar para mover/copiar
  • Menu de contexto com botão direito
  • Visualização de miniaturas

4. Software de Escritório: Exemplos e Aplicações

O que é Software de Escritório?

São aplicativos desenvolvidos para auxiliar em tarefas administrativas, comerciais e de produtividade em ambientes corporativos e educacionais.

Microsoft Word

Função: Processador de texto

Aplicações:

  • Redação de documentos
  • Relatórios empresariais
  • Cartas formais
  • Monografias e TCCs
  • Contratos

Exemplo de Exercício:

Criação de um currículo profissional com formatação adequada, incluindo seções para dados pessoais, experiência profissional e formação acadêmica.

Microsoft Excel

Função: Planilha eletrônica

Aplicações:

  • Controle financeiro
  • Análise de dados
  • Gráficos e relatórios
  • Orçamentos
  • Banco de dados simples

Exemplo de Exercício:

Criação de uma planilha de controle de gastos mensais com fórmulas para cálculo automático de totais, médias e criação de gráficos de pizza para visualização das despesas por categoria.

Microsoft PowerPoint

Função: Apresentações multimídia

Aplicações:

  • Apresentações comerciais
  • Aulas e palestras
  • Pitch de vendas
  • Relatórios visuais
  • Portfólios

Exemplo de Exercício:

Desenvolvimento de uma apresentação de 10 slides sobre um projeto empresarial, incluindo capa, sumário, slides de conteúdo com imagens e gráficos, e slide de conclusão com animações e transições profissionais.
