Lógica Computacional: A Arte do Pensamento Estruturado

A lógica computacional é o alicerce fundamental da programação e do desenvolvimento de sistemas. Ela nos ensina a pensar de forma estruturada, decompor problemas complexos e criar soluções eficientes através de algoritmos bem definidos.

1. O que são Fluxogramas? Simbologia Básica

O que são Fluxogramas?

Fluxogramas são diagramas que representam visualmente o fluxo de um processo ou algoritmo através de símbolos padronizados. Eles são ferramentas essenciais para planejar, documentar e comunicar processos de forma clara.

Simbologia Básica

Terminador: Início ou fim do processo

Processo: Operação ou ação

Decisão: Pergunta ou condição

Fluxo: Direção do processo

2. O que são Algoritmos? Onde são Usados?

Definição de Algoritmos

Algoritmos são sequências finitas de passos bem definidos e ordenados que resolvem um problema específico. Eles são a base de toda programação e processamento de dados.

Características de um Bom Algoritmo

  • Precisão: Passos claros e não ambíguos
  • Finitude: Deve terminar em algum momento
  • Entrada: Pode receber dados de entrada
  • Saída: Produz resultados
  • Efetividade: Cada passo deve ser executável

Onde são Usados?

Sistemas Operacionais

Gerenciamento de processos e memória

Redes Sociais

Algoritmos de recomendação de conteúdo

Navegação GPS

Cálculo de rotas mais eficientes

E-commerce

Sistemas de recomendação e busca

3. O que é Portugol? O que é Portugol Studio?

Portugol

Portugol é uma linguagem de programação algorítmica que utiliza vocabulário em português para facilitar o aprendizado da lógica de programação. É ideal para iniciantes que estão dando os primeiros passos na programação.

Portugol Studio

Ambiente de desenvolvimento integrado (IDE) específico para programação em Portugol. Oferece recursos como:

  • Editor de código com realce de sintaxe
  • Depurador integrado
  • Execução passo a passo
  • Verificação de erros em tempo real

4. O que são Variáveis e Constantes?

Variáveis

São espaços na memória do computador que armazenam dados que podem ser alterados durante a execução do programa.

// Declaração de variáveis em Portugol
inteiro idade
real altura
caracter letra
cadeia nome

Constantes

São valores fixos que não mudam durante a execução do programa. São definidos uma única vez e mantêm seu valor.

// Declaração de constantes em Portugol
constante real PI = 3.14159
constante inteiro DIAS_SEMANA = 7
constante cadeia NOME_EMPRESA = "Agência b-Look"

5. Tipos de Dados em Portugol

Booleano

Armazena valores lógicos: verdadeiro ou falso

logico estaChovendo = verdadeiro
logico ligado = falso

Caracter

Armazena um único caractere

caracter letra = 'A'
caracter simbolo = '#'

Cadeia (String)

Armazena sequências de caracteres (texto)

cadeia nome = "João Silva"
cadeia mensagem = "Olá, mundo!"

Inteiro

Armazena números inteiros sem casas decimais

inteiro idade = 25
inteiro quantidade = 100

Real

Armazena números com casas decimais

real altura = 1.75
real preco = 19.99

Vetor

Armazena coleções de elementos do mesmo tipo

// Vetor de 5 números inteiros
inteiro numeros[5] = {10, 20, 30, 40, 50}

// Vetor de nomes
cadeia nomes[3] = {"Ana", "Bruno", "Carla"}

6. Exemplos de Programas em Portugol com Funções

Exemplo 1: Calculadora Básica

programa
{
    funcao inicio()
    {
        real numero1, numero2, resultado
        cadeia operacao
        
        escreva("Digite o primeiro número: ")
        leia(numero1)
        escreva("Digite a operação (+, -, *, /): ")
        leia(operacao)
        escreva("Digite o segundo número: ")
        leia(numero2)
        
        se (operacao == "+") {
            resultado = numero1 + numero2
        } senao se (operacao == "-") {
            resultado = numero1 - numero2
        } senao se (operacao == "*") {
            resultado = numero1 * numero2
        } senao se (operacao == "/") {
            resultado = numero1 / numero2
        } senao {
            escreva("Operação inválida!\n")
            retorne
        }
        
        escreva("Resultado: ", resultado, "\n")
    }
}

Exemplo 2: Verificador de Idade

programa
{
    funcao inicio()
    {
        inteiro idade
        
        escreva("Digite sua idade: ")
        leia(idade)
        
        se (idade >= 18) {
            escreva("Você é maior de idade!\n")
        } senao {
            escreva("Você é menor de idade!\n")
        }
        
        se (idade >= 65) {
            escreva("Você é idoso!\n")
        }
    }
}

Exemplo 3: Contador Regressivo

programa
{
    funcao inicio()
    {
        inteiro numero, i
        
        escreva("Digite um número para contagem regressiva: ")
        leia(numero)
        
        para (i = numero; i >= 1; i--) {
            escreva(i, "...\n")
        }
        
        escreva("Fogo!\n")
    }
}

7. Exemplos de Programas em Portugol com Vetores

Exemplo 1: Média de Notas

programa
{
    funcao inicio()
    {
        real notas[5]
        real soma = 0.0, media
        inteiro i
        
        escreva("Digite 5 notas:\n")
        
        para (i = 0; i < 5; i++) {
            escreva("Nota ", i + 1, ": ")
            leia(notas[i])
            soma = soma + notas[i]
        }
        
        media = soma / 5
        escreva("Média das notas: ", media, "\n")
    }
}

Exemplo 2: Busca em Vetor

programa
{
    funcao inicio()
    {
        inteiro numeros[8] = {10, 25, 30, 45, 50, 65, 70, 85}
        inteiro busca, i
        logico encontrou = falso
        
        escreva("Digite um número para buscar: ")
        leia(busca)
        
        para (i = 0; i < 8; i++) {
            se (numeros[i] == busca) {
                escreva("Número encontrado na posição ", i, "\n")
                encontrou = verdadeiro
            }
        }
        
        se (nao encontrou) {
            escreva("Número não encontrado!\n")
        }
    }
}

Exemplo 3: Ordenação de Nomes

programa
{
    funcao inicio()
    {
        cadeia nomes[4] = {"Carlos", "Ana", "Bruno", "Diana"}
        cadeia temp
        inteiro i, j
        
        escreva("Nomes desordenados:\n")
        para (i = 0; i < 4; i++) {
            escreva(nomes[i], " ")
        }
        escreva("\n")
        
        // Ordenação por bolha
        para (i = 0; i < 3; i++) {
            para (j = i + 1; j < 4; j++) {
                se (nomes[i] > nomes[j]) {
                    temp = nomes[i]
                    nomes[i] = nomes[j]
                    nomes[j] = temp
                }
            }
        }
        
        escreva("Nomes ordenados:\n")
        para (i = 0; i < 4; i++) {
            escreva(nomes[i], " ")
        }
        escreva("\n")
    }
}
